# HTTP2 w/ Spring Boot + Undertow (Stripped-down footprint)

## Running from gradle

    $ ./gradlew run

## Creating and running uber-jar

    $ ./gradlew build
    $ java -Xbootclasspath/p:./lib/alpn-boot-8.1.6.v20151105.jar -jar build/libs/demo-0.0.1-SNAPSHOT.jar
